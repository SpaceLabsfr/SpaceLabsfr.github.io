Règles de confidentialité - Privacy Policy

Application Android JPO ITII localisation / GeolocJPO

Le présent document décrit et explique les règles de confidentialité de l’application JPO ITII localisation / GeolocJPO, dont l’utilité principale est de permettre la localisation des différents intervenants volontaires sur les stands de l’école d’ingénieurs ITII Normandie.

Cette application est uniquement à destination des intervenants (apprentis ingénieurs actuels, obligatoirement majeurs, volontaires pour présenter leur formation) lors des JPO de l’ITII Normandie.

Glossaire

JPO : Journée Porte Ouverte
ITII : Institut des Techniques d’Ingénieurs de l’Industrie
GPS : Global Positioning System = Système de géolocalisation mondial



Données collectées

Nom
Un prénom doit être utilisé pour identifier qui est l’intervenant sur la carte. Il est toutefois possible d’utiliser un pseudo. Cette information entrée par l’utilisateur sera transmise à notre base de données afin de l’associer aux coordonnées GPS transmises simultanément.

Position approximative
La position approximative de l’utilisateur peut être collectée s’il a désactivé la fonctionnalité de position exacte. L’application a cependant besoin de la position exacte pour être utile. Les données collectées sont la longitude, latitude et altitude de l’appareil de l’utilisateur qui seront associées au prénom ou pseudo renseigné.

Position exacte
La position exacte de l’utilisateur sera collectée, y compris en arrière-plan jusqu’à ce qu’il la désactive afin de permettre aux visiteurs de l’ITII lors des JPO de le localiser au sein de l’ITII. Les données collectées sont la longitude, latitude et altitude de l’appareil de l’utilisateur qui seront associées au prénom ou pseudo renseigné.

Remarque : Seules les données de géolocation capturées dans l'ITII Normandie, c'est-à-dire dans les limites suivantes, sont transmises.
49.107269 < latitude < 49.108780
1.496306 < longitude < 1.499954
Si l'utilisateur se trouve à l'extérieur de ces limites, ses coordonnées GPS ne seront pas transmises. Ce traitement est réalisé sur l'appareil de l'utilisateur.



Gestion des données

Transmission et stockage
Les données de géolocalisation sont transmises par internet à notre base de données Firebase Realtime Database.



Suppression des données

Comment supprimer les données concernant un utilisateur de l’application ? Une future fonctionnalité permettra à l’utilisateur de supprimer définitivement depuis l’application les données le concernant. Si un utilisateur de l’application souhaite que les données collectées soient totalement et définitivement supprimées, il peut envoyer un mail à kevindesir27@gmail.com. Sa requête sera traitée dans les meilleurs délais et il recevra un mail de réponse.



Partage des données à des tiers

Aucune information collectée avec l’application n’est transmise à des tiers. Elle n’est utilisée que pour l’usage auquel elle est dédiée (localisation sur les stands de l’ITII Normandie).
